🚀 COMEÇANDO EM 5 MINUTOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  cd natural-fruit-pwa
2️⃣  npm install
3️⃣  cp .env.example .env
    (Edite .env com credenciais Supabase)
4️⃣  Execute supabase-schema.sql no Supabase
5️⃣  Crie usuário admin no Supabase
6️⃣  npm run dev
7️⃣  Acesse http://localhost:5173

Detalhes: Veja INICIO-RAPIDO.md
