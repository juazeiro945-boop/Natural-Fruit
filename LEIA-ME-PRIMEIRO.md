## 🆘 PROBLEMAS?

### Erro ao instalar
```bash
rm -rf node_modules
npm install
```

### Erro no Supabase
- Verifique .env
- Confirme que executou o schema SQL
- Teste a URL: https://seu-projeto.supabase.co

### PWA não instala
- Use HTTPS em produção
- Vercel/Netlify fazem isso automaticamente
